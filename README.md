Enderman Cheers Wallet

Enderman Cheers Wallet é uma carteira digital minimalista e moderna para interações com Ethereum (ETH) e integração de pagamentos via PIX, permitindo mint direto de tokens JANUS26 com QR Code, utilizando Ethers.js para conexão com contratos inteligentes.

O projeto é desenvolvido com foco em UX simplificado, interface escura e responsiva, tornando a experiência de pagamento e minting mais intuitiva e rápida.

Funcionalidades

Conexão de carteira Ethereum via MetaMask.

PIX integrado com mint automático de tokens JANUS26.

Geração de QR Code para pagamentos instantâneos.

Logs em tempo real mostrando status das transações.

Layout minimalista: fundo escuro, elementos centralizados e contraste alto para melhor visualização.

Responsivo e otimizado para mobile.

Sistema de placeholders centralizados e inputs simplificados para rápida interação.

Tecnologias Utilizadas

HTML5 / CSS3: Estrutura e estilo da interface.

JavaScript: Lógica de interação, geração de QR Codes e comunicação com contratos inteligentes.

Ethers.js v6.8: Conexão com Ethereum, contratos e transações.

API de QR Code: Geração instantânea de QR Code para PIX.

Instalação e Uso

Clone o repositório:

git clone https://github.com/seuusuario/enderman-cheers-wallet.git


Abra o arquivo HTML no navegador moderno (recomendado Chrome ou Edge).

Conectar a carteira Ethereum:

Clique em Conectar Carteira.

MetaMask deve estar instalada e desbloqueada.

Informe o endereço do contrato JANUS26 quando solicitado.

Gerar QR Code e Mint:

Preencha o valor em BRL, a chave PIX e observação (opcional).

Clique em Gerar QR Code + Mint.

O QR Code será exibido e a transação será enviada automaticamente para o contrato.

Acompanhar logs:

Verifique os logs em tempo real no painel inferior da wallet.

Observações

Certifique-se de usar uma carteira MetaMask configurada na rede correta do contrato JANUS26.

Valores PIX são apenas para geração de QR Code; a conversão e minting ocorrem via contrato inteligente.

Todos os placeholders e inputs são centralizados para melhor UX e acessibilidade.

O layout é otimizado para dispositivos móveis e desktops.
